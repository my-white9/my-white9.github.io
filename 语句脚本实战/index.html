if语句

判断一个服务是否正常

#!/bin/env bash
read -p service service
systemctl status $service |grep running &> /dev/null
if [ $? -eq 0 ];then
echo “$service excited”
else
echo “$service not excited”
fi

判断用户是否存在

#!/bin/env bash
#Name: user excited

read -p "user:" user
id $user &> /dev/null
if [ $? -eq 0 ] ;then
    echo "$user has excited"
else
    echo "$user not excited"
fi

判断软件包是否安装

#!/bin/env bash
#Name: user excited

read -p "rpm" rpm
rpm -qa |grep $rpm  &> /dev/null
if [ $? -eq 0 ] ;then
    echo "$rpm has excited"
else
    echo "$rpm not excited"
fi

判断当前主机的内核版本

#!/bin/env bash
number=$(uname -r)
version1=${number:0:1}
version2=${number:2:1}
if [ $version1 -eq 2 -a $version2 -ge 6 ] ;then
    echo "$number"
else
    echo "no"
fi

批量创建用户

需求：批量加5个新用户，以u1到u5命名，并统一加一个新组，组名为class,统一改密码为123

#!/bin/env bash
Name: create user
grep -w ^class /etc/group &>/dev/null
test $? -ne 0 && groupadd class
for ((i=1;i<=5;i++))
do
    useradd -G class u$i
    echo 123 |passwd --stdin u$i
done

局域网内脚本检查主机网络通讯

需求2：

写一个脚本，局域网内，把能ping通的IP和不能ping通的IP分类，并保存到两个文本文件里

以10.1.1.1~10.1.1.10为例

#!/bin/env bash
for ip in {1..10}
do
    ip=192.168.0.$ip
    ping -c1 $ip &>/dev/null
if [ $? -eq 0 ] ;then
    echo "$ip" >>  /ip.txt
else
       echo "$ip" >> /ip1.txt
fi
done

脚本同步系统时间

① 具体需求
写一个脚本，30秒同步一次系统时间，时间同步服务器192.168.0.139
如果同步失败，则进行邮件报警,每次失败都报警
如果同步成功,也进行邮件通知,但是成功100次才通知一次

1.死循环
2.同步命令 ntpdate 192.168.0.1
3.同步成功计数，条件到达100报警
#!/bin/env bash
i=0
while true
do
ntpdate 114.118.7.161 &>/dev/null
if [ $? -eq 0 ] ;then
let i++
if [ $i -eq 100 ];then
echo “system date success”| mail -s “success” root@localhost
let i=0
else
echo “system date false”|mail -s “false” root@localhost
echo “count:”$i
fi
fi
sleep 3
done

㈠ 具体需求 ##
使用until语句批量创建10个用户，要求stu1—stu5用户的UID分别为1001—1005；
stu6~stu10用户的家目录分别在/rhome/stu6—/rhome/stu10

创建10个用户条件为真推出

#!/bin/env bash
ls /rhome &>/dev/null
if [ $? -ne  0 ];then
    mkdir /rhome
else
    echo 'excited'
fi
count=0
until [ $count -gt 10 ]
do
    for x in {1..10}
    do
        if [ $x -ge 6 ];then
            let count++
               useradd -d /rhome/stu$x stu$x
        else
            let count++
            useradd -u 100$x stu$x
        fi
    done
done
